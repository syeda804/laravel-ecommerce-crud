# Laravel eCommerce CRUD

A complete **Laravel-based eCommerce CRUD application** built to demonstrate modern full-stack web development concepts using the Laravel MVC architecture. This project includes product management, category handling, user authentication, and complete Create, Read, Update, Delete operations with a clean and responsive interface.

---

## Features

* User Authentication (Login / Register)
* Product Management
* Category Management
* Full CRUD Operations
* Clean Laravel MVC Structure
* Responsive Admin Interface
* Database Relationships (One-to-Many / Foreign Keys)
* Validation & Error Handling
* Easy to Customize and Extend

## Installation Guide

### 1. Clone Repository

```bash
git clone https://github.com/your-username/laravel-ecommerce-crud.git
cd laravel-ecommerce-crud
```

### 2. Install Dependencies

```bash
composer install
npm install
npm run dev
```

### 3. Configure Environment

Copy `.env.example` file:

```bash
cp .env.example .env
```

Update database credentials inside `.env`

### 4. Generate Application Key

```bash
php artisan key:generate
```

### 5. Run Migration

```bash
php artisan migrate
```

### 6. Start Server

```bash
php artisan serve
```

Open in browser:

```text
http://127.0.0.1:8000
```
